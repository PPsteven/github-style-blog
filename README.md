# PPSteven's Blog

A personal blog about technology, coding, and everything in between. Built with [Hugo](https://gohugo.io/) and the [github-style](https://github.com/MeiK2333/github-style) theme.

## About This Blog

This blog documents my explorations in software engineering, DevOps, home server setup, and algorithmic problem solving. Posts are written in Markdown and generated as static HTML for fast, secure hosting on GitHub Pages.

## Quick Start

```bash
hugo server --buildDrafts
```

Visit http://localhost:1313/

## Writing Posts

```bash
hugo new post/my-new-post.md
```

Add `<!--more-->` in your Markdown content to control what appears in post previews.

## Theme

Uses [MeiK2333/github-style](https://github.com/MeiK2333/github-style), a Hugo theme that makes static blogs visually resemble GitHub's interface.

## Deploy

```bash
hugo
```

The generated site lives in the `public/` directory, ready to be pushed to GitHub Pages or any static hosting provider.
