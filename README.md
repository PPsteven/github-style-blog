# github-style-blog

A Hugo blog with GitHub-style theme, migrated from CodePainter/blog.

## Quick Start

```bash
hugo server --buildDrafts
```

Visit http://localhost:1313/

## Build for Production

```bash
hugo
```

## Writing Posts

```bash
hugo new post/my-new-post.md
```

## Theme

Uses [MeiK2333/github-style](https://github.com/MeiK2333/github-style)
